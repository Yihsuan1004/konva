要为`Konva`的形状设置线连接，我们可以在实例化形状时设置`lineJoin`属性，或者我们可以使用`lineJoin（）`方法。

`lineJoin`属性可以设置为尖角，斜角或圆。 除非另有说明，否则默认线连接为尖角。

![](images/linejoin.png)  
说明：鼠标经过三角形以更改其线连接样式。  

Konva Line Join Demo[点击查看](https://konvajs.github.io/downloads/code/styling/Line_Join.html)   


    <!DOCTYPE html>
    <html>
    <head>
    <script src="https://cdn.rawgit.com/konvajs/konva/1.4.0/konva.min.js"></script>
    <meta charset="utf-8">
    <title>Konva Line Join Demo</title>
    <style>
    body {
      margin: 0;
      padding: 0;
      overflow: hidden;
      background-color: #F0F0F0;
    }
    </style>
    </head>
    <body>
    <div id="container"></div>
    <script>
        var width = window.innerWidth;
        var height = window.innerHeight;
        var stage = new Konva.Stage({
            container: 'container',
            width: width,
            height: height
        });
        var layer = new Konva.Layer();
        var triangle = new Konva.RegularPolygon({
            x: stage.getWidth() / 2,
            y: stage.getHeight() / 2,
            sides: 3,
            radius: 70,
            fill: 'red',
            stroke: 'black',
            strokeWidth: 20,
            lineJoin: 'bevel'
        });
        triangle.on('mouseover', function() {
            this.lineJoin('round');
            layer.draw();
        });
        triangle.on('mouseout', function() {
            this.lineJoin('bevel');
            layer.draw();
        });
        // add the shape to the layer
        layer.add(triangle);
        // add the layer to the stage
        stage.add(layer);
     </script>
    </body>
    </html>
